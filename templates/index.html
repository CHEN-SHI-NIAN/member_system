Background Introduction

本專案是一個市場監控工具，旨在實時監控以太坊智能合約的行為。用戶可通過輸入合約地址和 ABI，輕鬆獲取合約的可調用函數。

Key Features

1. 帳密比對: 比對帳號密碼並對密碼進行加密。
2. ABI解析: 提取輸入的 ABI 中的可調用函數。
3. 即時結果: 提交數據後即時顯示處理結果和函數列表。
4. 友好界面: 簡潔的前端設計，便於操作。
5. 錯誤處理: 提供錯誤提示，幫助用戶識別問題。

How to Use

1. 訪問頁面: 先註冊帳號並登入進入會員頁面。
2. 輸入地址和 ABI: 分別填寫合約地址和 ABI。
3. 提交數據: 點擊 "Submit" 按鈕。
4. 查看結果: 檢查處理結果和函數列表。
5. 進一步查詢: 可選擇查詢合約詳細信息或返回主頁。
